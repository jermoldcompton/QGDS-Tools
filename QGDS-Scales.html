QGDS Scales

QGDS Scales is an interactive reference tool for exploring the mathematical relationships between the three core token scales in the Queensland Government Design System: Grid, Size, and Spacing. These scales share an exponential structure with calibrated divergence rates, each tuned to the perceptual demands of its role.

Scale relationship

Grid scale grows slowly as the common denominator; Size and Spacing diverge progressively faster.
Export chart
Size and Spacing both start at 2× the Grid increment (a=4 vs a=2) and pull further ahead as their faster growth intervals accumulate. All three scales share the same stepped exponential structure, diverging by design, not by accident.
Each scale's step size stays flat, then doubles at its interval. The doubling intervals are in the ratio 4 : 2 : 1, so each scale doubles exactly twice as fast as the one above it.

Scale explorer

Each scale follows the formula, where the base increment, growth factor, and doubling interval can be adjusted independently. The tool visualises how the three scales diverge over their range, demonstrates why their parameters were chosen, and allows practitioners to explore the effect of changing any variable in real time. Use it to audit token values, validate scale decisions, export reference data for spreadsheets, or communicate the underlying structure of the system to stakeholders and engineers.

Formula:
Sₙ = Sₙ₋₁ + a × b^⌊(n−1)/c⌋

where:
S = start value, a = base increment, b = growth factor, c = interval.
